# AUto-data-science
# AutoML Pro 🤖
End-to-end Automated Machine Learning web app built with Streamlit.

## Features
- 10+ ML models (XGBoost, LightGBM, CatBoost, RF, GB...)
- Auto hyperparameter tuning (GridSearchCV)
- SHAP explainability
- ARIMA time series forecasting
- Clustering (K-Means, DBSCAN, GMM)
- PCA / SVD dimensionality reduction
- Claude AI report generation
- PDF export

## Live Demo
[Open App](https://auto-data-science.streamlit.app/)

## Tech Stack
Python · Streamlit · Scikit-learn · XGBoost · LightGBM · Matplotlib · SHAP · Anthropic API
